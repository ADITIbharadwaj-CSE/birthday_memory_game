
# 🎂 Birthday Memory Game

A fun and emotional memory-based web game built using personal moments 💖  
This project combines creativity with coding to create an interactive experience.

---

## 🌐 Live Demo
👉 https://aditibharadwaj-cse.github.io/birthday_memory_game/

---

## 🚀 Features
- 🧠 Memory-based card matching game
- 🎨 Attractive UI with smooth design
- 📸 Personal photos used as game cards
- 🔄 Restart option anytime
- 📱 Fully responsive (works on mobile & desktop)

---

## 🛠️ Tech Stack
- HTML5 → Structure of the game
- CSS3 → Styling, layout, and animations
- JavaScript → Game logic and interactions

---

## ⚙️ How This Project Works
- Cards are displayed in a grid format
- Each card hides an image
- User clicks to flip cards
- If two cards match → they stay open
- If not → they flip back
- Game continues until all pairs are matched
